# TradeTrack 📈

TradeTrack is a modern Android application designed to help traders log, analyze, and improve their trading performance through structured journaling, performance tracking, and insightful analytics.

## 🚀 Features

- **Trade Logging**  
  Record every trade with details such as:
  - Entry price
  - Exit price
  - Stop loss
  - Take profit
  - Position size
  - Trade direction (Buy/Sell)
  - Risk-to-Reward ratio

- **Trading Journal**  
  Add notes, strategies used, emotions, and lessons learned for each trade.

- **Performance Analytics**  
  Track key trading metrics such as:
  - Win rate
  - Profit/Loss
  - Average risk-reward ratio
  - Best-performing strategies
  - Trading consistency

- **Dashboard Overview**  
  Get a quick summary of trading performance through a clean and intuitive dashboard.

- **User Authentication**  
  Secure login and account creation for personalized tracking.

- **Onboarding Guide**  
  Interactive guide for new users to understand how the app works.
